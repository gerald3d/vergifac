==TVA Codes et taux
Suppression d'un code et de son taux:
 _Selectionner la ligne du code + taux dans la boite déroulante. Clic sur 'Enlever'(Remove), 
 _Enregistrer (Save) Code et taux sont supprimés.
Créer un nouveau code et son taux:
_Clic sur 'Ajouter'(Add).
_Saisie du code (1 caractère alphabétique) , saisie du taux en pourcentage avec éventuellement des décimales (marque décimale est le point '.').
_Si 'Saisie OK' Enregistrer.
Modifier un taux en gardant le même code,
_ il faut supprimer la ligne du code + (ancien) taux et recréer le code avec le nouveau taux.
Vérifier que chaque code taxe soit unique.

Si besoin, il est possible de définir un code TVA avec un taux de 0%.



==Paramètres divers
1_Fichier de commande de mise en page par défaut.
   bouton choix: liste les fichiers commande de mise en page. Choisir le fichier par défaut utilisé pour éditer les factures.
2_Nombre maximum de lignes de facture.
   Suivant la taille courante de vos factures choisir le nombre maximum de lignes (qui consomme de la mémoire et ralentit le programme).
La prise en compte de ce paramètre se fait à l'activation de la fenêtre 'Saisie Facture'.
Modifier ce paramètre fenêtre 'Saisie Facture' fermée.
Si votre document en cours a besoin de plus de lignes, pensez à 'Sauv-doc' pour sauver votre document afin de pouvoir le relire après modification du nombre de lignes.
Ne pas oublier d'enregistrer.

==Création d'un nouveau dossier
Création de votre dossier pour mémoriser principalement vos clients et produits.
Choisir un nom qui vous parle, il vous sera proposé à chaque démarrage de Vergifac bouton 'Ouvrir un dossier'.
Nom: xxxxxxx.db.  par exemple si votre enseigne est 'JOJO SARL' ,  nom= jojosarl.db.
Pour accéder à ce paramètre, il faut ouvrir un dossier (dbtestfac.db par exemple).
 
Usage de Vergifac

Le répertoire actif doit être celui où ce trouve le programme vergifac.
Ceci permet au programme de trouver sans problème:

-le ou les dossiers base de données  ( pour le test: dbtestfac.db)

Les répertoires associés:

  _doc-aide contient ce manuel d'aide. Ne pas toucher.

  _histo-fac contient les factures définitives que vous avez sauvées après l'impression.
   (Vous pouvez supprimer les factures de test)
   C'est dans ce répertoire qu'il faut aller chercher vos factures pour les (re)imprimer ou les joindre à un mail.

 _memo-doc  contient les saisies sauvées.
  Vous pouvez supprimer les fichiers tests.

 _misenpage contient les fichiers de commande de la mise en page.
  Conseil: conserver les fichiers de démonstration. En faire des copies et les renommer avant de les modifier pour coder votre propre mise en page.

 _outils. Ne pas toucher. Permet la création de dossier.
